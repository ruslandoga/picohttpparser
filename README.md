# PicoHTTPParser

[h2o/picohttpparser](https://github.com/h2o/picohttpparser) bindings for Elixir.

## Installation

The package can be installed by adding `picohttpparser` to your list of dependencies in `mix.exs`

```elixir
def deps do
  [
    {:picohttpparser, "~> 0.1.0"}
  ]
end
```
